## sklearn (机器学习)

** sklearn (scikit-learn [官网](http://scikit-learn.org/stable/)) 是机器学习中一个常用的python模块，依赖于Numpy，SciPy 和 matplotlib. **

sklearn 包含了很多种机器学习的方式：

- 分类 (Classification)  
- 回归 (Regression) 
- 非监督分类 (Clustering) 
- 数据降维 (Dimensionality reduction) 
- 模型选择 (Model Selection) 
- 数据预处理 (Preprocessing)
